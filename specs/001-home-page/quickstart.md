# Quickstart: Build eShop home page with navigation, product list, filters, and footer

## Prerequisites

- Node.js 18+
- npm or yarn or pnpm

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) to view the home page

## Features

- **Navigation**: Logo, search bar, user profile icon
- **Product Display**: Grid/list of products with images and details
- **Filtering**: By type, price range, branch
- **Sorting**: By price (low to high, high to low)
- **Responsive**: Works on desktop, tablet, and mobile

## Development

- Edit components in `/components/`
- Update data in `/lib/data/products.ts`
- Add types in `/types/index.ts`
- Run tests: `npm test`
